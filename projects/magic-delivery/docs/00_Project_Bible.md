# 项目总设定

## 一句话故事

一个现实世界的外卖员接到“魔王城订单”，被异世界众人误认为救世勇者，最后靠外卖配送逻辑拯救世界。

## 核心笑点

史诗世界观和外卖员职业病互相撞车：

- 预言石碑被当成订单备注。
- 魔法阵被当成骑手路线规划。
- 龙族契约被解释成平台会员协议。
- 勇者小队被改造成配送站。
- 魔王毁灭世界的理由只是连续 999 次外卖无人接单。

## 主角定位

阿伟不是传统勇者。他的能力来自职业习惯：

- 路线规划
- 催单抗压
- 客诉处理
- 扫码付款
- 电动车极限走位
- “还有三分钟超时”的生存本能

## 制作原则

1. 镜头以“可复用素材 + 可编辑合成”为主。
2. 动作用姿势替换、表情差分、镜头运动和特效贴片表达。
3. AI 视频/AI 生图用于补充高动感镜头、角色新姿势、背景氛围和特效素材。
4. 人工主要审剧情节奏、素材选择和镜头方向；细节错误交给校验脚本和资产锚点表。
